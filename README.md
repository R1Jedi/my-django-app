🛍️ Store Server

Django интернет-магазин с полным функционалом электронной коммерции.

✨ Функциональность

🎯 Основные возможности \
• Каталог товаров - категории, фильтрация, поиск \
• Корзина покупок - добавление/удаление товаров \
• Аутентификация - регистрация, вход, верификация email \
• Восстановление пароля - полный цикл сброса через email \
• Онлайн-платежи - интеграция с Stripe \
• Управление заказами - история заказов, статусы \
• Email уведомления - различные типы уведомлений \
• Админ-панель - полное управление магазином

🛠 Технические особенности \
• Redis - кэширование и сессии \
• PostgreSQL - основная база данных \
• Stripe API - обработка платежей \
• Django-Allauth - система аутентификации с подтверждением email \
• Celery - асинхронная отправка email для подтверждения аккаунта \
• Django Debug Toolbar - отладка разработки

🔐 Система аутентификации

👤 Функционал пользователя \
• Регистрация с подтверждением email (через Celery) \
• Вход/выход из системы \
• Восстановление пароля через email \
• Смена пароля \
• Подтверждение email адреса \
• Управление профилем пользователя

🔑 Восстановление пароля \
Полный цикл сброса пароля через email: \
• Запрос на восстановление \
• Отправка письма с ссылкой \
• Подтверждение токена \
• Установка нового пароля \
• Уведомление об успешном изменении

📧 Система email уведомлений

⚡ Отправка через Celery \
• Подтверждение регистрации - верификация email адреса (асинхронно через Celery)

📨 Обычная отправка (синхронно) \
• Восстановление пароля - ссылка для сброса пароля \
• Создание заказа - подтверждение оформления заказа \
• Оплата заказа - уведомление об успешной оплате \
• Смена статуса - изменение статуса заказа
